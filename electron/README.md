# Юки — Запуск как Electron-приложение (Windows 10)

## Что понадобится
- **Node.js 18+** → [nodejs.org](https://nodejs.org)
- **Git** → [git-scm.com](https://git-scm.com)

---

## Быстрый старт

### Шаг 1 — Собери веб-часть (из корня проекта)
```bash
npm install
npm run build
```

### Шаг 2 — Запусти Electron
```bash
cd electron
npm install
npm run dev:win
```

Юки появится в правом нижнем углу **поверх всех окон**.

---

## Собрать .exe установщик

```bash
cd electron
npm install
npm run build
```

Файл появится в `electron/dist-electron/` — устанавливается как обычная программа.

## Голосовые команды

| Команда | Действие |
|---|---|
| «Открой браузер» | Запускает Chrome |
| «Следующая вкладка» | Ctrl+Tab |
| «Предыдущая вкладка» | Ctrl+Shift+Tab |
| «Закрой вкладку» | Ctrl+W |
| «Открой Telegram» | Telegram Desktop |
| «Открой Discord» | Discord |
| «Прочитай сообщения» | Вкладка чатов |
| «Запусти музыку» | Яндекс Музыка |
| «Открой настройки» | Параметры Windows |
| «Открой терминал» | Командная строка |

## Управление окном

- **Перетащить** — зажми заголовок «Юки» и тяни мышью
- `—` — свернуть виджет
- `⌄` — свернуть окно в трей
- `✕` — закрыть приложение
